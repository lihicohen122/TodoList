1. Install this chart (required first):
   helm install my-release oci://ghcr.io/lihicohen122/todolist/todolist --set database.password=<YOUR-PASSWORD>
   
   Or with a specific version (current: {{ .Chart.Version }}):
   helm install my-release oci://ghcr.io/lihicohen122/todolist/todolist:{{ .Chart.Version }} --set database.password=<YOUR-PASSWORD>

2. Start the tunnel (required so the app is reachable):
   minikube tunnel
   (Leave this running in a separate terminal. On Windows you may need to run as Administrator.)

3. Access the application:
   http://127.0.0.1

4. API endpoints:
   - GET/POST /todos - Backend API
   - /webui - WebUI interface
   - / - Frontend application

5. Check pod status:
   kubectl get pods

6. View application logs:
   kubectl logs -l app={{ .Values.backend.name }}
   kubectl logs -l app={{ .Values.frontend.name }}

7. To uninstall:
   helm uninstall {{ .Release.Name }}
